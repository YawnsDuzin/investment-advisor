- [x] 공개 API parity: 13개 심볼 시그니처 완전 일치 (`tools/a_db_split/capture_api.py` 기반)
- [x] `analyzer/*`, `api/*`, `shared/logger.py` 무수정 동작 — 9개 외부 모듈 import 성공
- [x] `shared.db` 최상위에서 private 심볼 (`_migrate_to_v*`, `_validate_proposal`, `_create_base_schema`) 접근 불가 (`__all__` 차폐)
- [x] `pytest tests/test_tier_limits.py` 21개 전원 통과 (v16 마이그레이션 테스트 포함)
- [x] `pytest tests/test_admin_tier_audit.py` 29/35 통과 — 실패 6건은 A-트랙 이전(B3 리팩터) 시점부터 존재하는 기존 결함으로 확인 (사전 베이스라인 commit `260e2fe` 체크아웃 후 동일 실패 재현)
- [x] Fresh DB에서 `init_db()` 실행 후 `pg_dump --schema-only` diff 확인:
  - 스키마 DDL 본문 2,440줄 **완전 동일** (byte-identical)
  - diff 출력: `\restrict` / `\unrestrict` 보안 토큰 2줄만 차이 (pg_dump가 매 실행마다 생성하는 랜덤 세션 토큰 — DDL 아님)
- [x] `init_db()` idempotent — 기존 DB에 재실행 시 no-op
- [x] 마이그레이션 가드 — `run_migrations()`가 dict에 없는 버전 요청 시 `RuntimeError` 발생
- [x] `api.main` import 성공 (스모크 테스트 통과)

## 1. 문서 목적·요약

본 문서는 3-tier 구독 요금제 출시를 위해 비즈니스 검토 보고서에서 식별한 **모든 "작업이 필요한 부분"**을 추출하여, 실제 실행 가능한 Action Item 단위로 분해한 **실행 계획서**이다. 구체적인 코드 설계나 비즈니스 의사결정 재논의는 포함하지 않는다.

- **총 Action Item**: 31개
- **P0 (출시 차단)**: 17개 / **P1 (출시 1개월 내)**: 10개 / **P2 (출시 후 확장)**: 4개
- **핵심 마일스톤**: Phase 0 완료 시 클로즈드 베타 개시 가능
- **추적성**: 각 항목은 원천 보고서의 섹션을 `(원천: §X.Y)` 형태로 명시

### 카테고리 범례

| 아이콘 | 카테고리 | 설명 |
|:------:|----------|------|
| 🛠️ | Dev (Backend) | DB 스키마, API, 배치 로직 변경 |
| 🎨 | Dev (Frontend) | Jinja2 템플릿, UI/UX 개선 |
